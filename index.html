Criar um formulário com validação

https://www.figma.com/file/zBKnYG9UNdUiIr8ClQTWSG/?node-id=3%3A2

Pontos de atenção:

  • A validação do formulário deve ser feita com javascript puro.
  • Ao clicar para enviar o formulário, se caso algum campo não estiver preenchido, a borda do input deve ficar vermelha e uma mensagem de "campo obrigatório" deve aparecer embaixo do campo que não foi preenchido, conforme o figma.
  • O fundo do formulário deve ser feito usando a imagem em anexo na aula.

O que foi feito:

  • Foi criada uma verificação antes do envio do formulário para verificar se todos os campos estão preenchidos, ou seja, o formulário só é enviado caso todos os campos estejam preenchidos.
  • Foi criada uma validação após o campo ser preenchido, o que deixa o campo com as bordas verdes. Caso as informações preenchidas no campo sejam apagadas, o campo recebe a mensagem "campo obrigatório" e as bordas ficam vermelhas, também acontece o mesmo se clicar no campo não preencher nada e simplesmente remover o foco do mesmo.
  • Foi adicionada uma máscara para telefone utilizando uma biblioteca, mas não foi feito nenhum tipo de validação ou tratamento no campo de email.
  • Não foi criado nenhum modal para ser exibido após o envio do formulário.
*campos obrigatórios
